# Peptide Vial Label Generator

A simple browser-based tool for creating custom labels for 3ml peptide vials.
Designed to fit 20 × 40mm labels and print on Niimbot or compatible label printers.

## How to use

1. Enter your **brand / lab name** — appears at the top of the label.
2. Fill in **manufacturer, peptide name, size,** and **date**.
3. Optionally add a **URL** and customize the **footer note**.
4. Click **Generate Label**, then **Download JPEG**.
5. Print at 100% scale on a 20 × 40mm label.
